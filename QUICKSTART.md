# Quick Start Guide

## Installation

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your MongoDB URL and JWT secret
npm run dev
```

### Frontend
```bash
cd frontend
npm install
cp .env.local.example .env.local
npm run dev
```

## Modules Included

- **Auth**: Fully functional

Visit http://localhost:3000 to get started!
